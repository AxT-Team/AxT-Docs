让我们先来了解一下项目结构

```bash 项目结构
AxTBot-Public
├─.venv   <-- 虚拟环境

├─logs    <-- 日志目录（可更改）
├─plugins <-- 插件目录
├─src     <-- 源码目录
│  ├─app
│  │  ├─__init__.py
│  │  ├─routes.py
│  │  └─exceptions.py
│  └──Utils
│     ├─__init__.py
│     ├─AutoUpdate.py
│     ├─Config.py
│     ├─ConfigClass.py
│     ├─ConfigCli.py
│     ├─Database.py
│     ├─EventClass.py
│     ├─Logger.py
│     └─Processer.py
├─main.py  <-- 入口文件
├─config.yaml  <-- 配置文件
└─pyproject.toml  <-- 项目文件
```
