# 查询IP的详细信息

用法：`@AxT社区机器人 /ipinfo [IP地址]`

示例：

```
@AxT社区机器人 /ipinfo 8.8.8.8
```

返回示例：

```
=====IP信息=====
IP: 8.8.8.8
| 开始 IP: 8.8.8.8
| 结束 IP: 8.8.8.8
| 归属地: 美国加利福尼亚州圣克拉拉县山景市
| 纬度: 37.42301
| 经度: -122.083352
| ISP: 谷歌公司DNS服务器
| LLC: GOOGLE
| ASN: AS15169
==============
```
