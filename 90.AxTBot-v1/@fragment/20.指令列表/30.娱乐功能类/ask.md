# 赛博答案之书

用法：`@AxT社区机器人 /ask [疑问]`

示例：

```
@AxT社区机器人 /ask 我今天会幸运吗
```

返回示例：

```
要求不要太高。
```

::: tip Tips:
答案仅限娱乐，请勿当真.
:::
