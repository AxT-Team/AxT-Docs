# 获取指定IP的延迟（Ping）信息

用法：`@机器人 /ping [域名或IP] [节点名]`

|可选节点|节点参数|
| ------------ | ------------ |
| cn | 中国湖北十堰/电信 |
| hk | 中国香港/腾讯云 |

示例：
```
@机器人 /ping 8.8.8.8 hk
```

返回示例：
```
=====Ping信息=====
主机名: 8.8.8.8
| IP: 8.8.8.8
| 最大延迟: 2.82 ms
| 平均延迟: 2.766 ms
| 最小延迟: 2.73 ms
| 归属地: 美国加利福尼亚州圣克拉拉县山景市谷歌公司DNS服务器
| 检测点: 中国香港/腾讯云
==============
```
