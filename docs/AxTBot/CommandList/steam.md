# 查询Steam信息数据

用法：`@机器人 /steam [ID/昵称]`

示例：

```
@机器人 /steam NoNameGMM
```

返回示例：

```
====Steam账户信息====
| 社区资料状态：公开
| 用户名：NoNameGMM
| 真实姓名：NoNameGMM
| Steam ID：76561199445846334
| 账户创建日期：2022-12-19 07:54:29
| 最后下线日期：1970-01-01 08:00:00
| 地理位置：CN
=====================
```
